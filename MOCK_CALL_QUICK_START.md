# Mock Call New - Quick Start Guide

## 🎉 100% Free Training

Mock Call New is **completely free** - no API costs at all!

## Quick Setup

### 1. Get OpenRouter API Key (Free)
```
1. Visit https://openrouter.ai/keys
2. Sign up (free, no credit card)
3. Click "Create Key"
4. Copy your API key
```

### 2. Add API Key to Database
```sql
INSERT INTO app_secrets (key, value)
VALUES ('openrouter_api_key', 'sk-or-v1-YOUR_KEY_HERE');
```

### 3. That's It!
- No OpenAI API key needed
- No credits to add
- No payment required
- Ready to use!

## How to Use

### Text-Based Training
1. Go to `/mock-calls-new`
2. Select a scenario
3. Click "Start Call"
4. Type your messages
5. Get instant AI responses
6. Complete and view evaluation
**Cost: $0.00**

### Voice Training (Optional)
1. Enable voice toggle
2. Click microphone button
3. Speak your message
4. AI responds with voice
5. Complete and evaluate
**Cost: $0.00**
