### Access
Navigate to: `/mock-calls-new`

- ✅ **Unlimited practice** - No limits whatsoever
- ✅ **Repeat any scenario** - Practice same scenario multiple times
- ✅ **100% Free** - Zero API costs
- ✅ **AI**: OpenRouter Llama 3.1 8B (free model)

- Daily practice

## Original Mock Calls - LIMITED PREMIUM

### Access
Navigate to: `/mock-calls`

### Features
- ✅ **Premium AI**: OpenAI GPT-4o
- ✅ **Premium Voice**: OpenAI Whisper & TTS-1
- ✅ **High Quality**: Better responses and evaluations
